# Topology Files

This directory contains the Cisco Packet Tracer simulation files used to build and test the network design.

## Planned Contents

| File | Description |
|------|-------------|
| baseline_network.pkt | Simulation of the current flat network (before any changes) |
| secured_network.pkt | Simulation of the final design with QoS, Port Security, VLANs, and DHCP Snooping |

To open these files, Cisco Packet Tracer version 8.0 or later is required.
