# Topology Files

This folder contains the Packet Tracer simulation files for the project.

| File | Description |
|------|-------------|
| baseline_network.pkt | The original flat network before any changes |
| secured_network.pkt | The final setup with QoS, Port Security, VLANs, and DHCP Snooping working |

You need Cisco Packet Tracer 8.0 or newer to open these files.
